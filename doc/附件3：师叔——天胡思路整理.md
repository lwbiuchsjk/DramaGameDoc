## 总体思路

**摁下葫芦又起瓢。怀念过去，展望未来。坚定、乐天的理想主义者。一个乐天派的小老头。一个物欲很强的人。一个禁不住诱惑的人。**
+ 一个物欲很强的人为什么会选择出世？
	他的决定一开始获得支持。之后没有人支持。之后他发现自己的决定会导致不好的事情发生。之后他再否定自己。之后他再选择了一条道路，并坚持下去。一开始，他是入世的坚定支持者。之后，他发现应该出世。

面对已经转动的历史车轮，门派中的六位长老，和其他门派的人物都有自己的想法。各怀心思。

最重要的是，师叔面对必将到来的大潮，自己所做的思考，以及坚定的决心和信念。师叔面对的考研是，如何做一个艰难的决定。当自己做了错误的决定之后，如何改正自己。如何承担责任。

### 细致地梳理

天胡做的事情是什么？他在努力推进同帝国的合作。新训改革也是在为同帝国合作做准备。因为这样的合作可以为门派带来更多的资源，才能让门派生存下去。为什么？因为门派日益发展壮大，成为中原地区施法者的培养基地。在这种情况下，施法者的日常生活训练培养开销日渐强烈。因此，门派才不得不为了养育这么多施法者而寻求出路。
+ 现在施法者与过去施法者生活的对比。

天胡在努力推进门派、联盟帝国的合作。在这个过程中，他遇到的阻碍是什么？
起初，天胡所处的政治环境如下。
+ 掌教
	支持。希望未来的施法者有一个更好的环境，能够不被他人欺凌。能够不要以自己是施法者为耻。
+ 外勤部
	反对。十分怀念以前施法者遗世独立的情况。因为自己童年的遭遇，所以一直对凡人心怀憎恨。之所以继续作斩妖除魔的事情，除了是因为自己只会这么做，还因为这是同门所托。既然别人托付给自己，自己就理应承担。荣誉感很强。
+ 内务部
	支持。因为天胡是自己的师兄，所以为了他好才支持他。“如果我不支持你，就没有人支持你。”但是另一方面，站在天胡的立场上、为了他好，她也可以做出一些强迫、要求的行为。为了他好，而能跨过很多底线。
+ 戒律堂
	观望。在更高的层面上不知道该作何决定。究竟是应该同贪婪的凡人合作，还是坚守施法者的本分遗世独立？
+ 法术部
	无所谓。将研究视为一切。为了研究可以不择手段。年少时想加入鲁丘的门派，但是因为天资不足被拒绝。对此耿耿于怀。因此对鲁丘门派地弃徒们抱有比较复杂的感情。一方面他们都没有再做研究的工作，鄙视他们。另一方面，自己又嫉妒他们曾经的经历。
	认为天胡所做的工作是没有意义的。
+ 雨晴
	无所谓。因为自己门派深陷尸术研究的泥潭，所以没有太多功夫去考虑未来。但是在她眼中，凡人就是羔羊。自己要保护羔羊，但是自己也会去吃羔羊。自己要守护苍生，但是苍生也会为自己的研究做出牺牲。
+ 饕郭
	支持。因为自己门派与帝国也有合作，所以支持。但是他并不知道自己合作的对象是鹰派而不是鸽派。
+ 尔古
	支持。施法者与凡人的另一种相处方式。相依为命。十分单纯，只是抱着合作的心态前来。
+ 鲁丘
	不明。在他的眼中所有的路都是绝路。所以他对自己收下的弟子不闻不问。

### 姓名的来历

“我爸没什么文化，成天就知道瞎混。年纪轻轻的时候有了我和我弟。哦对了，还没跟你说我有个弟弟。我叫天，他叫地，我爹姓胡。他觉得两个孩子胡天胡地特别像他的生活。”

“结果呢？”

“三岁那年，我弟弟因为没钱看病死了。五岁那年，我爹因为喝酒太多再也没醒来。”

“你妈呢？”

“四岁的时候，知道我能使用灵力的第二天，我妈再也没有回来。”

“所以我痛恨自己的出身。但是这个姓……毕竟是爹娘给的。只是我不想把这个字放在前面。所以我叫天胡。这倒也好。十四张牌放在桌子上，推到就是赢家。”
